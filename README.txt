Product Boxes
Written by: Roy K.
Last Modified: 2021-05-13

HTML
- Create 3 product boxes
- Each box will contain an image, a title and a button
- Create a popup with an overlay
CSS
The popup should be hidden by default.
- If the popup is open:
● The overlay should be gray
● The popup should appear in middle of the page
● The popup size should be width: 50%; height: 50%
JS
- Clicking on a box's button will open the popup
- When the popup is opened, fetch a description related to the clicked box from the server and show it as the popup content
- Add a way to close the popup
PHP
- Create an abstract class to handle AJAX requests
- Use the abstract class to handle the HTTP requests
